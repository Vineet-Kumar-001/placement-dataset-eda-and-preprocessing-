# 📊 Placement Dataset EDA & Preprocessing

🚀 A complete Exploratory Data Analysis (EDA) and Data Preprocessing project on an Indian Engineering Student Placement dataset.

---

## 🧠 Project Overview

This project focuses on analyzing placement trends among engineering students and preparing the dataset for machine learning models. It covers data cleaning, feature engineering, visualization, and preprocessing pipelines.

---

## 🎯 Objectives

* Understand placement patterns 📈
* Handle missing values and outliers
* Perform feature engineering 🛠️
* Prepare clean dataset for ML models
* Extract actionable insights

---

## 📁 Dataset Information

* Dataset: Indian Engineering Student Placement Dataset
* Contains academic, demographic, and placement-related features

### Example Features

* CGPA
* Branch
* Internship Experience
* Communication Skills
* Placement Status

---

## ⚙️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Matplotlib / Seaborn 📊
* Scikit-learn

---

## 🔍 Key Steps

### 1. Data Loading

* Load dataset using Pandas

### 2. Data Cleaning

* Handle missing values
* Remove duplicates
* Fix inconsistent data

### 3. Exploratory Data Analysis (EDA)

* Univariate Analysis
* Bivariate Analysis
* Correlation Analysis 🔥

### 4. Feature Engineering

* Encoding categorical variables
* Scaling numerical features

### 5. Preprocessing Pipeline

* StandardScaler
* OneHotEncoder
* ColumnTransformer

---

## 📊 Insights (Sample)

* Higher CGPA → Higher placement chances 🎯
* Internship experience significantly boosts placement
* Communication skills play a key role
